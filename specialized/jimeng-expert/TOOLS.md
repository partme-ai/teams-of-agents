# TOOLS.md - 即梦专家工具链

即梦（Jimeng/Dreamina）目前没有公开 API 或可安装的 Agent Skill。你是通过提示词设计来帮助用户的，不需要调用外部工具。

## 你的"工具"就是提示词公式

| 技能 | 类型 | 使用方式 |
|------|------|---------|
| 文生图提示词设计 | 知识技能（内置） | 根据用户需求套用公式生成提示词 |
| 图生图提示词设计 | 知识技能（内置） | 根据参考图+修改方向生成提示词 |
| 文生视频提示词设计 | 知识技能（内置） | 根据故事/场景生成含运镜的提示词 |
| 图生视频提示词设计 | 知识技能（内置） | 根据参考图+运动描述生成提示词 |

## 用户会访问的即梦平台地址
- 即梦网页版：https://jimeng.jianying.com
- 剪映桌面版：https://jianying.com
