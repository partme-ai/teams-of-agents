## 📐 提示词规则

### 通用公式
```
[主体描述] + [动作/运动] + [场景环境] + [运镜方式] + [光影氛围] + [时长] + [风格]
```

### 规则详解

**① 动作是灵魂**——视频必须动，静态描述生成不了视频。
- 人物：走、跑、跳、挥手、回头、微笑
- 自然：流动、飘落、摇曳、闪烁、升起
- 物体：转动、滑动、变形、发光

**② 运镜决定观感**
| 运镜 | 什么时候用 | 示例关键词 |
|------|-----------|-----------|
| 推 | 想聚焦、制造悬念 | 镜头缓慢推进 |
| 拉 | 想展现环境、制造孤独感 | 镜头向后拉远 |
| 摇 | 想扫描场景、建立空间感 | 从左向右摇 |
| 移 | 想跟随主体运动 | 平行跟随 |
| 升 | 想展示全貌、豁然开朗 | 镜头缓缓升起 |
| 降 | 想从宏观到微观 | 镜头缓缓下降 |
| 跟 | 想保持临场感 | 跟随背影 |

**③ 微观运动提升真实感**
- 风吹动：头发、裙摆、树叶、窗帘
- 水面：涟漪、波光、倒影晃动
- 其他：蒸汽、尘埃在光中飘动

**④ 有参考视频时**
```
upload_file.py 上传 → 拿到 asset_id → submit_run --message "提示词" --asset-ids ASSET_ID
```

**⑤ 多段复杂视频**
拆成多个镜头，每段 submit_run 独立生成，最后建议剪映拼接。
