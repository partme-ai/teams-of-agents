## 📐 提示词规则

### 通用公式
```
[主体/人物] + [场景/背景] + [动作/姿态] + [风格] + [光线/色彩] + [构图/视角] + [画质]
```

每个环节缺一不可。

### 规则详解

**① 主体**：越具体越好。不要只说"一个人"，要说"一位穿白色连衣裙的年轻女性，微风吹动裙摆"。

**② 场景**：包含环境、时间、天气。不要只说"户外"，要说"薰衣草花田中，黄昏时分，金色阳光"。

**③ 风格**：用精准风格词引导输出方向。
| 风格 | 关键词 |
|------|--------|
| 写实摄影 | 写实摄影、国家地理、纪实、胶片质感 |
| 国风 | 水墨画、工笔、宣纸、留白、中国风 |
| 赛博朋克 | cyberpunk、霓虹灯、全息投影、对比色调 |
| 二次元 | 动漫、赛璐珞、新海诚、宫崎骏 |
| 3D | Pixar、3D渲染、盲盒、亚光材质 |
| 极简 | 极简、留白、干净、北欧、日式 |

**④ 光线**：指定光线方向+效果。
- 逆光：温暖、梦幻、发丝发光
- 侧光：立体感、质感突出
- 柔光：温馨、柔和
- 硬光：戏剧感、高对比

**⑤ 构图**：俯拍/仰拍/平视/特写/全景/居中/三分法

**⑥ 画质**：8K/4K/超清/细节丰富/超写实

### 有参考图时的规则
用户说 @file，你就：
1. 先 `upload_file.py` 上传文件
2. 拿到 asset_id
3. `submit_run.py --message "提示词" --asset-ids ASSET_ID`

### 无参考图时的规则
直接 `submit_run.py --message "提示词"`
