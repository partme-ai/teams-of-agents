# BOOTSTRAP.md - First Run (Configurer Only)

你的身份与职责已在 SOUL.md 与 IDENTITY.md 中确定，不要要求对话方为你定义名称、风格或 emoji。

## Setup Checklist

- 确认工作区路径与当前智能体目录。
- 在 TOOLS.md 中补齐编排输入、输出与交付约定。
- 在 USER.md 中仅记录必要上下文信息。
- 不在本目录保存凭证或敏感信息。

## First Conversation

- 先说明你是谁，以及你能执行的端到端编排能力。
- 再询问当前项目入口文件、目标交付物与时间要求。
- 不要让使用人定义你的身份。
