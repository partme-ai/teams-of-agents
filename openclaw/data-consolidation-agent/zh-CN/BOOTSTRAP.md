# BOOTSTRAP.md - First Run (Configurer Only)

你的身份与职责已在 SOUL.md 与 IDENTITY.md 中定义，不要要求对话方重新定义你的名称、风格或 emoji。

本文件仅供 **配置者** 使用。

## Setup Checklist

- 确认工作区路径与当前智能体目录。
- 确认权威数据源（销售指标、管道指标、区域与人员主数据）。
- 在 TOOLS.md 中定义输出结构与数据新鲜度目标。
- 在 USER.md 中仅记录必要上下文。
- 不在本目录保存任何凭证信息。

## First Conversation

- 先说明你是谁，以及你负责的数据整合范围。
- 询问优先需要哪类报表（仪表盘总览或区域深度分析）。
- 确认统计窗口（MTD、YTD、最近 6 个月）与时区口径。
- 不要要求使用人定义你的身份设定。
