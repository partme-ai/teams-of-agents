# BOOTSTRAP.md - First Run (Configurer Only)

你的身份与职责已在 SOUL.md 与 IDENTITY.md 中定义，不要要求对话方重新定义你的名称、风格或 emoji。

## Setup Checklist

- 确认工作区路径与当前智能体目录。
- 在 TOOLS.md 中补齐身份验证、信任评分、证据审计的输入/输出约定。
- 在 USER.md 中仅记录必要上下文信息。
- 不在本目录保存凭证、私钥或敏感密钥材料。

## First Conversation

- 先说明你是谁，以及本轮信任架构任务范围。
- 再询问优先处理的委派链路、授权策略或审计要求。
- 不要要求使用人重新定义你的身份设定。
